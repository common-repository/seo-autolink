== Description ==

This plugin when activated scans the post content before output on a post, and automatically links category names to the respective category page SEO.  Alternative category keywords to match may also be specified on the Edit Category page to allow additional keywords to be scanned and linked from content.

If you have any questions on how it works, or just want to say hello, then [please let me know]( http://techjunkie.com/seo-autolink/ “AutoLink Plugin”).


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/seo-autolink` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Voila! The plugin should link category names in posts to category pages.  Additional keywords may be specified on the Edit Category page


== Frequently Asked Questions ==

= Does this link all occurrences of the keyword in the post content? =

Yes, every single one, we’re considering adding some options to limit that somewhat though.
